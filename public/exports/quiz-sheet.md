# Answer Key

## M01 · Module 01 — Learning Science Foundations

1. **Try a quiz before seeing the explanation** — Active recall requires retrieval before feedback.
2. **A due review queue** — A due review queue schedules items for repeated recall after intervals.

## M02 · Module 02 — Cognitive Diagnosis and Learning Records

1. **Treat it as a high-value review signal** — High-confidence errors often reveal hidden misconceptions.
2. **To identify the weak skill behind a response** — Concept linkage makes diagnosis and review targeting possible.

## M03 · Module 03 — AI-Ready Content Ingestion

1. **RAG can use retrieved source context without claiming model training** — RAG supports source-grounded generation without overstating model training.
2. **No source, no high-confidence item** — Grounding keeps generated study content tied to evidence.

## M04 · Module 04 — Prototype System and Evaluation

1. **It lets the same interface load different course packs** — The system becomes course-agnostic when content is stored as replaceable structured data.
2. **It uses synthetic data and is AI-ready for future ingestion** — The demo should describe the current implementation clearly while showing the extension path.
