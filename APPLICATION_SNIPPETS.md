## CV project entry

**Modular Exam Revision Hub**  
Self-initiated React/Vite prototype for modular online exam preparation.

- Built an interactive revision interface with React, Vite, React Router, KaTeX, and modular JavaScript content packs.
- Designed a modular content data model for concepts, quizzes, common traps, formulas, figures, long-answer prompts, and printable review sheets.
- Prototyped the original private version with my own revision materials, then replaced the content with synthetic sample data for the public release to avoid sharing private academic materials.
- Implemented prototype learning-support features such as spaced review, mistake tracking, priority dashboard, search, and local browser progress records.
- Used learning-science ideas such as spaced repetition, active recall, cognitive-load reduction, and mistake-based review to guide the interface design.
- Planned future extension with LLM/RAG-based course-content ingestion, knowledge tracing, and cognitive diagnosis.
